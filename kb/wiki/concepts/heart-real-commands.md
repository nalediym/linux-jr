# Heart: Real Linux Commands

> **First seen in:** [story.yaml](../sources/linux-jr-story--story.yaml.md)
> **Also referenced by:** [commands.md](../sources/linux-jr-story--world--commands.md.md), [ethos.md](../sources/linux-jr-story--world--ethos.md.md), [educational.md](../sources/linux-jr-story--world--educational.md.md), [CLAUDE.md](../sources/CLAUDE.md.md)
> **Confidence:** high (5 sources agree)
> **Aliases:** THE HEART, Track A, real Linux

## Definition
The non-negotiable core of Linux Jr: a kid who completes Season 1 can type the real shell commands `pwd`, `ls`, `cd`, `cat`, `mkdir`, `touch`, `echo`, and `help` in a real terminal — not a simulated dialect — and understands what each does.
— [commands.md](../sources/linux-jr-story--world--commands.md.md)#c-c6ef427f

## Details
Everything else in the bible (ethos, pedagogy, setting, characters) is in service of this one outcome.
— [story.yaml](../sources/linux-jr-story--story.yaml.md)#c-77916829, [commands.md](../sources/linux-jr-story--world--commands.md.md)#c-55674787

The heart has a strict tiebreaker: if ethos, pedagogy, or narrative ever conflicts with "real commands taught by real typing," the heart wins.
— [story.yaml](../sources/linux-jr-story--story.yaml.md)#c-77916829, [commands.md](../sources/linux-jr-story--world--commands.md.md)#c-55674787

Four never-compromise commitments: (1) commands are real; (2) typing is mandatory (no click-to-execute); (3) the player graduates — by M10 kid can open a real Terminal and navigate their own Mac; (4) no gamified shortcuts at the cost of muscle memory.
— [story.yaml](../sources/linux-jr-story--story.yaml.md)#c-77916829, [commands.md](../sources/linux-jr-story--world--commands.md.md)#c-f1f3317e

Pretend languages (Scratch, drag-and-drop, fake robot DSLs) teach logic but do not build muscle memory. Linux Jr solves this: a kid who finishes 10 missions has already typed `ls` hundreds of times.
— [commands.md](../sources/linux-jr-story--world--commands.md.md)#c-c97c193b

Designed constraints: real filesystem semantics, real errors with kid-friendly wording, behavioral task checks (any real-Linux path works), tab completion, autocorrect/autocapitalize OFF on iPad. CLAUDE.md preserves these invariants in code.
— [commands.md](../sources/linux-jr-story--world--commands.md.md)#c-6efe10b6, [CLAUDE.md](../sources/CLAUDE.md.md)#c-ac29fd74
