# Signature Phrases

> **First seen in:** [story.yaml](../sources/linux-jr-story--story.yaml.md)
> **Also referenced by:** [style.md](../sources/linux-jr-story--world--style.md.md), [pip.yaml](../sources/linux-jr-story--characters--pip.yaml.md), [captain-rex.yaml](../sources/linux-jr-story--characters--captain-rex.yaml.md), [sprocket.yaml](../sources/linux-jr-story--characters--sprocket.yaml.md)
> **Confidence:** high (5 sources agree)

## Definition
Fixed catchphrases that give each character a recognizable voice. Used consistently across mission copy and TTS.
— [style.md](../sources/linux-jr-story--world--style.md.md)#c-d622fd21

## Details
- **Pip:** "Oh! Oh! Watch this!" (starting something new), "Okay okay okay —" (thinking aloud).
— [pip.yaml](../sources/linux-jr-story--characters--pip.yaml.md)#c-53a2d278
- **Captain Rex:** "Let's have a look." (briefing), "Well now." (surprised), "Mm." (thinking). Never raises his voice.
— [captain-rex.yaml](../sources/linux-jr-story--characters--captain-rex.yaml.md)#c-35b48b21
- **Sprocket:** only beeps — "Beep-boop!", "Beep." (yes/hi), "Beep-beep-beep!" (excited/found it), long beep for "oh no".
— [sprocket.yaml](../sources/linux-jr-story--characters--sprocket.yaml.md)#c-2507fc2e

story.yaml and style.md both enumerate the same set of signature phrases, so they're canonical.
— [story.yaml](../sources/linux-jr-story--story.yaml.md)#c-d9dfbe60, [style.md](../sources/linux-jr-story--world--style.md.md)#c-d622fd21
